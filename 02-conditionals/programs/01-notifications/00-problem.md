# Temperature-Based Notification System

## Problem Statement

Create a simple notification system that displays messages based on the current temperature.  
The system helps users understand weather conditions and take appropriate action.

## Task

- Take a single input representing the temperature (in degrees Celsius).
- Based on the temperature value:
  - If temperature is **less than 15**, display **"Cold weather"**
  - If temperature is **between 15 and 30 (inclusive)**, display **"Normal weather"**
  - If temperature is **greater than 30**, display **"Hot weather"**

## Input Format

- A single integer representing temperature in Celsius

## Output Format

- Display a notification message based on the temperature
