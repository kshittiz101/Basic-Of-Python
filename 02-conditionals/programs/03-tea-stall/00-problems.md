# Tea Stall Cup Pricing Program

## Problem Statement

A tea stall offers tea in different cup sizes, and each size has a fixed price.  
You are required to write a program that calculates and displays the price of tea based on the cup size entered by the user.

## Cup Sizes and Prices

| Cup Size | Price |
|---------|-------|
| small   | 10    |
| medium  | 20    |
| large   | 30    |

## Task

- Take a single input representing the cup size.
- If the input is:
  - `"small"` → display price **10**
  - `"medium"` → display price **20**
  - `"large"` → display price **30**
- For any other input, display **"Invalid input"**.

## Input Format

- A single string representing the cup size  
  (e.g., `small`, `medium`, `large`)

## Output Format

- Display the price of the tea if the input is valid  
- Otherwise, display `Invalid input`
